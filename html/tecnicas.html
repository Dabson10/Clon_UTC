Tecnicas de aprendizaje

Aprendizaje supervisado

El modelo aprende a partir de datos etiquetados, donde cada entrada tiene una salida esperada. Se usa para clasificación y regresión.
  • Regresión Lineal y Logística: Modelos matemáticos para predicción.
  • Máquinas de Soporte Vectorial (SVM): Clasificadores eficientes para datos complejos.
  • Redes Neuronales Artificiales (ANNs): Modelos inspirados en el cerebro humano.
  • Árboles de Decisión y Random Forest: Algoritmos basados en reglas para clasificación.
  • K-Vecinos Más Cercanos (KNN): Clasifica datos según su cercanía a otros.

Aprendizaje No Supervisado

El modelo analiza datos sin etiquetar y encuentra patrones o estructuras por sí solo. Se usa para agrupamiento y reducción de dimensionalidad.
  • K-Means: Agrupa datos en clusters según similitud.
  • DBSCAN: Detecta grupos de datos con densidad similar.
  • Algoritmos de Asociación (Apriori, FP-Growth): Encuentran relaciones entre datos
  • Análisis de Componentes Principales (PCA): Reduce la cantidad de variables manteniendo la información clave

Aprendizaje Por Refuerzo

El modelo aprende a tomar decisiones a través de un sistema de recompensas y castigos. Se usa en robótica, videojuegos y optimización de sistemas.
  • Q-Learning: Aprende a través de prueba y error sin un modelo del entorno.
  • Deep Q-Networks (DQN): Usa redes neuronales para mejorar el Q-Learning.
  • Algoritmos basados en Políticas (REINFORCE, PPO, A3C): Optimizan directamente las acciones para maximizar recompensas.
